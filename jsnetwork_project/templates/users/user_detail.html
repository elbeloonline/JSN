How to use this site

This page provides general instructions for how to use this site.

Click the orders button to be taken to your list of previous orders. From here you can click the 'Add Order' link to create a new order. Fill in the information you have available for the name search. The Client name, a client file number and a person name OR company name are required.

The remaining fields are all optional. Dates are automatically set to the last 20 years from today.

Once the order is entered and you have verified your entry, you are taken back to the orders list.

To run the search, click the refresh icon beside the button labeled 'FULL'. This will take you to a screen that refreshes while the names are being searched. This screen will automatically return you to the order list once the order is complete. The button will be blue instead of gray when the order is finished generating.

If you want to enter an additional name search while the current search is being executed, you can click on the link to go back to the orders screen and enter another. This won't affect your current search.

If multiple searches are being caried out there may be orders in the queue waiting to be serviced. You can check the status of the queue by click on the searches backlog (coming soon!).

Clicking on the client file number will let you view details about the information you entered for a particular order.
